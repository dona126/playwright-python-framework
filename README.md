**Playwright Python Automation Framework** 
---
UI automation framework built using Python, Pytest, and Playwright following the Page Object Model (POM) design pattern.

**Tech Stack**
---
1. Python
2. Pytest
3. Playwright

**Features**
---
1. Login validation
2. Product validation
3. Add to cart testing
4. Cart verification
5. POM structure
6. Playwright assertions

**Project Structure**
---
```
pages/   -> contains page classes, locators, and page actions
tests/   -> contains test cases and validations/assertions
